Over deze Website

Deze portfolio website heb ik gemaakt om mijn vaardigheden en projecten als software developer te tonen. De website is gebouwd met behulp van bootstrap, JavaScript, HTML en CSS. HTML wordt gebruikt voor de structuur waarin ook Bootstrap staart die ervoor zorgt dat de website responsive is en op elk apparaat kan worden gebruikt. JavaScript en CSS zorgen ervoor dat er animatie's te zien zijn bij het open van de website of pagina.

Kenmerken:

  • Responsive design voor alle schermformaten
  • Contactgetgevens voor communicatie
  • CV download mogelijkheid
